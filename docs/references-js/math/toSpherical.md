### Description

Gets the spherical coordinates form the provided space.

This function is commonly used with [getSpace()](/references-js/input/getSpace.html).

### Syntax
```js
toSpherical(space);
```

### Parameters
**space** Vec3: coordinate space

### Returns
Vec3: spherical coordinate space

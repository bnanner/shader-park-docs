### Description
Mix between two materials. Works with [color()](/references-js/color/color.html), [metal()](/references-js/color/metal.html), and [shine()](/references-js/color/shine.html)

### Syntax
```js
mixMat(amount);
```

### Parameters
**amount** Number: value from 0 to 1.0 to mix
